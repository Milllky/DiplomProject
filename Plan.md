### 5. Тестирование производительности
- Время загрузки приложения

## Тестовое окружение:
- Ручное тестирование проводится на устройстве Honor 50 с ОС Android 13
- Автоматизирование тестирование производится на ПК с ОС Windows 10, при помощи приложения Android Studio с инструментами Espresso и Allure
- Для автоматизированного тестирования необходимо эмулируемое устройство с Android API не ниже 29

## Тестовая документация:
- На основе плана составляется Чек-лист проверок
- Составляются тест-кейсы для ручных и автоматизированных проверок
- Автоматизированные проверки будут заканчиваться с готовыми автоматическими отчётами
