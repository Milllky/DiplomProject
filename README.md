# Запуск авто-тестов

## Перед запуском:

- Установить Android Studio
- Клонировать репозиторий
- Подготовить эмулятор
    Установить язык устройства - русский

## Запуск проекта:

### Открыть проекта "fmh_android_15_03_24" в Android Studio

## Запуск тестов:

Запуск всех тестов сразу можно осуществить двумя способами:

- В папке .run открыть файл AllTests.run.xml
- Через вкладку Run запустить их вручную

## После тестов:

### Скопировать отчёт Allure

- Создать в корне проекта папку "allure-results"
- На тестируемом устройстве найти папку "/data/data/ru.iteco.fmhandroid/files/allure-results" и сохранить из неё все файлы в только что созданную папку в проекте

### Открыть отчёт:

- Через терминал подать команду: "allure serve" !НЕОБХОДИМО НАХОДИТСЯ В КОРНЕ ПРОЕКТА!
- Отчёт откроется в браузере
- После работы с отчётом необходимо закрыть терминал
